eleccionesdb

Base de datos relacional abierta con resultados electorales de España: elecciones generales al Congreso y elecciones autonómicas de 9 comunidades autónomas. Cubre desde 1980 hasta 2023 y ofrece datos a nivel de provincia, municipio y sección censal.

Descargar datos → Ver modelo de datos

¿Qué contiene?

🗳️

10 ámbitos electorales

Congreso + 9 comunidades autónomas: Andalucía, Aragón, Asturias, Baleares, Canarias, Castilla y León, Cataluña, Comunidad Valenciana y Comunidad de Madrid.

📅

~40 años de datos

Cobertura temporal desde 1980 (Cataluña) hasta 2023, con todas las convocatorias de elecciones generales y autonómicas disponibles.

📦

3 formatos de descarga

Disponible en Parquet (análisis eficiente), SQLite (esquema relacional completo) y CSV planos (pre-joineados, listos para usar).

Estructura de la base de datos

Esquema estrella con tablas de dimensiones y hechos, diseñado para análisis electoral.

📐

5 dimensiones

Tipos de elección, elecciones, territorios (con jerarquía CCAA → provincia → municipio → sección), recodificaciones de partidos y partidos.

📊

2 tablas de hechos

Resumen territorial (censo, participación, votos válidos/blancos/nulos) y votos territoriales (votos y representantes electos por partido).

✈️

2 tablas CERA

Resumen y votos de residentes ausentes (Censo de Españoles Residentes Ausentes).

Ver modelo completo →

Formatos de descarga

Los datos están disponibles en tres formatos, pensados para diferentes perfiles de uso.

⚡

Parquet

Tablas normalizadas en formato columnar. Ideal para R, Python y DuckDB.

🗄️

SQLite

Esquema relacional completo con PKs, FKs e índices. Consultas SQL directas.

📄

CSV planos

Tablas pre-joineadas listas para Excel, Google Sheets o importación directa.

Ir a descargas →

Acceso programático

¿Prefieres consultar los datos mediante una API? EleccionesDB API ofrece acceso REST gratuito, abierto y sin autenticación a toda la base de datos.

🔌

API REST abierta

Endpoints para elecciones, territorios, partidos y resultados. Filtros flexibles, paginación y respuestas JSON.

🚀

Sin autenticación

Acceso directo desde cualquier lenguaje: curl, Python, R, JavaScript. Sin claves, sin registro.
